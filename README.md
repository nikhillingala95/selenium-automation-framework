# selenium-automation-framework
Automation testing framework built using Selenium WebDriver and Java.

**Tools Used**
* Java
* Selenium WebDriver
* TestNG

**Expectations**
* Automated login test
* Form submission testing
* Cross-browser testing (basic)

**Path Flow**
* Clone repo
* Open in Eclipse/IntelliJ
* Run TestNG.xml

**Outcome**
Reduced manual testing effort and improved efficiency
